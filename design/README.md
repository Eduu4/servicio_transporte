- logo_demo.html — vista previa local del logo y paleta

Instrucciones para Figma:
1. Abre Figma, crea un nuevo archivo y arrastra los SVGs desde tu máquina al canvas.
2. Ajusta tipografías (Inter/Roboto) y exporta los assets si quieres/los necesites para PNG.
3. Crea frames para variantes (monocromo, claro/oscuro) y prototipo si lo deseas.
4. Para compartir públicamente: en Figma, selecciona "Share" → "Anyone with the link" → "Can view" y copia el link.

Si quieres, puedo preparar un ZIP con estos archivos o generar versiones adicionales (monocromo, favicon, 512x512 PNG).
